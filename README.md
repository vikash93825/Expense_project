* Sign Up, Login Page 
  - Basic validation check for email & password
* Dashboard page 
  - Add new transaction, transaction details.
* Ledger Page 
  - Filter according to transaction type.
 

## INTRODUCTION:
* It is an application that helps you manage and track your daily expenses.
